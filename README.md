# 🧾 Landing Page – Rock Band (Events and Promotion)


-------------------------------------------------------------------------

## 📌 Description

This page is a landing page for a rock band you want promote your identity, music and upcoming events. The goal is to transmit energy, style and personality, managing to capture the attention of potential attendees at his concerts.


Technologies Used

- HTML
- CSS

Structure

- index.html

- style.css: Contains the styles used throughout the page except for the header and footer. This imports two CSS files located in the folder containing all the CSS files, called "css".

- album.css

- sobre.css

- header.css

- footer.css

##  ▶️ What we can see on the page
When the program starts, a navigation bar appears at the top, allowing the user to navigate to different sections of the page.

The page content reflects:

- Event information (dates, venues, city).
- The band's visual identity.
- A style consistent with the musical genre.
- Use of striking and coherent images.

It also includes:

- Musical influences.
- The band's values.
- Relevant social media links.


## Page style

- A color palette in keeping with the rock style.
- Typography with personality.
- Use of Flexbox and/or Grid Layout.
- A striking and organized visual design.
- 
The page also has a responsive design, allowing access from different devices.
